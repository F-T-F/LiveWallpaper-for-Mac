<h2> 簡介 </h2>
簡單易用的 macOS 動態桌布應用程式，可將影片設為桌面背景。程式於選單列（Menu Bar）執行，不會顯示於 Dock。支援影片選擇、靜音切換、多螢幕顯示，
並會自動記住上次使用的影片及靜音設定
